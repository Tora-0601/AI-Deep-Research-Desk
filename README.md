# 🔍 AI Deep Research Desk

**AI Deep Research Desk** は、Google Gemini API を活用して、複雑なトピックを深く調査・分析するためのリサーチ支援ツールです。

制作チーム：**WA・NAO・SA（ワナオサ）**

---

## 🚀 主な機能
- **AI プロンプト自動生成**: 調査したいタイトルと納期、トピックを入力するだけで、Gemini が最適なリサーチ用プロンプトを作成します。
- **ディープリサーチ実行**: 大規模言語モデル（Gemini 2.5/2.0）を使用し、構造化された詳細な調査レポートを生成します。
- **依頼履歴の保存**: SQLite データベースを搭載し、過去のリサーチ結果をいつでも振り返ることができます。
- **BYOK (Bring Your Own Key)**: 各ユーザーが自身の Google AI API キーを入力して使用する安全な設計です。
